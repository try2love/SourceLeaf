# Third-party licenses

SourceLeaf currently links only Apple system frameworks and Swift standard-library components at build time.

Optional external programs are discovered or installed separately and remain under their own licenses:

- Codex CLI — provided by OpenAI under its applicable terms.
- Tectonic — optional managed LaTeX engine; license information must be displayed before installation.
- MacTeX, TinyTeX, TeX Live, latexmk, BibTeX, and Biber — optional external toolchains under their respective licenses.

This file must be updated whenever code, binaries, fonts, icons, or other assets are added.
